## How to Add New Documentation

1. **Via Dify Workflow**: Use HTTP POST requests to automatically create new pages
2. **Manual Creation**: Add MDX files directly to the appropriate directory
3. **Automatic Navigation**: New pages automatically appear in the sidebar

## Directory Structure

- **incident-management/**: Emergency procedures, troubleshooting guides
- **root-cause-analysis/**: Investigation templates, analysis frameworks  
- **process-improvement/**: Optimization guides, workflow enhancements
- **strategic-planning/**: Infrastructure plans, capacity management
- **documentation-workflow/**: Standards, automation, knowledge management
